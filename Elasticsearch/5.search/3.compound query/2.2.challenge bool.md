## 2Ô∏è‚É£ Challenge Questions

### üîπ Challenge 1: Find all available books.

- Retrieve documents where `category` is "Books" and `status` is "available".

### üîπ Challenge 2: Find products with high stock and discount.

- Retrieve documents where `stock` is `>= 10` and `discount` is `true`.

### üîπ Challenge 3: Exclude expensive electronics.

- Retrieve documents where `category` is "Electronics" but exclude items where `price` is greater than `1000`.

### üîπ Challenge 4: Find books and cheap electronics.

- Retrieve documents where `category` is "Books" **OR** `category` is "Electronics" **AND** `price` is less than `500`.

### üîπ Challenge 5: Best deals.

- Retrieve documents where `discount` is `true`, `stock` is greater than `0`, and `price` is between `100` and `500`.

## 3Ô∏è‚É£ Practice Queries

### ‚úÖ Solution for Challenge 1

```json
GET my_bool_index/_search
{
  "query": {
    "bool": {
      "must": [
        { "term": { "category": "Books" } }
      ],
      "filter": [
        { "term": { "status": "available" } }
      ]
    }
  }
}
```

### ‚úÖ Solution for Challenge 2

```json
GET my_bool_index/_search
{
  "query": {
    "bool": {
      "filter": [
        { "range": { "stock": { "gte": 10 } } },
        { "term": { "discount": true } }
      ]
    }
  }
}
```

### ‚úÖ Solution for Challenge 3

```json
GET my_bool_index/_search
{
  "query": {
    "bool": {
      "must": [
        { "term": { "category": "Electronics" } }
      ],
      "must_not": [
        { "range": { "price": { "gt": 1000 } } }
      ]
    }
  }
}
```

### ‚úÖ Solution for Challenge 4

```json
GET my_bool_index/_search
{
  "query": {
    "bool": {
      "should": [
        { "term": { "category": "Books" } },
        { "bool": {
            "must": [
              { "term": { "category": "Electronics" } },
              { "range": { "price": { "lt": 500 } } }
            ]
        }}
      ],
      "minimum_should_match": 1
    }
  }
}
```

### ‚úÖ Solution for Challenge 5

```json

        { "term": { "discount": true } }
      ],
      "filter": [
        { "range": { "stock": { "gt": 0 } } },
        { "range": { "price": { "gte": 100, "lte": 500 } } }
      ]
    }
  }
```
