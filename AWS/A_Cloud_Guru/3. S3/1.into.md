| Version ID | Quality | Object URL                                                            |

| `v3mno`    | 1080p   | `https://my-bucket.s3.amazonaws.com/videos/movie.mp4?versionId=v3mno` |
| `v4pqr`    | 4K      | `https://my-bucket.s3.amazonaws.com/videos/movie.mp4?versionId=v4pqr` |
